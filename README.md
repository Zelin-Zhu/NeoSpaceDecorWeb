# Neospace Decor Website

这是一个现代家居装饰网站的静态网站项目。

## 项目结构

```
my-decor-site/
├── index.html          # 主页面（包含导航、轮播图框架、产品列表、页脚）
├── css/
│   └── style.css       # 大牌感配色、字体定义和布局样式
├── js/
│   └── main.js         # 轮播图逻辑、导航栏滚动变色等交互
├── assets/             # 存放所有静态资源
│   ├── logo.svg        # 公司矢量Logo
│   ├── slider/         # 存放轮播图高清大图
│   └── products/       # 存放家具分类图片（沙发、桌椅等）
├── favicon.ico         # 浏览器标签栏的小图标
└── README.md           # 项目说明文档
```

## 功能特性

- **响应式设计**: 适配桌面和移动设备
- **轮播图**: 自动播放的主页轮播图
- **导航栏**: 滚动时变色的固定导航栏
- **产品展示**: 网格布局的产品列表
- **现代UI**: 大牌感配色和优雅的字体

## 如何使用

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html`
3. 或者使用本地服务器运行（推荐）

## 自定义

- 在 `assets/slider/` 中添加轮播图图片
- 在 `assets/products/` 中添加产品图片
- 修改 `css/style.css` 调整样式
- 编辑 `js/main.js` 添加更多交互

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
